# Tests

Run from project root: `pytest tests/ -v` or `python -m pytest tests/ -v`.

Current scope: unit and component tests (with mocks). There are no live end-to-end integration tests (e.g. Supervisor → real Qdrant) yet; contributions in that direction are welcome.

Full test documentation (how to run, what to add, data requirements): **[docs/testing/](../docs/testing/)**.
